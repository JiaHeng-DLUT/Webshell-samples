将 HTML 文件上传到您的服务器
如果您可以将新文件上传到自己的网站，则选择此项。		 操作说明：
1. 新建空白网页，复制下面的字符串，将其粘贴至空白网页并保存，命名为site-verification.html
<{$token}>
2. 将该文件上传到 <{$siteurl}>
3. 通过在浏览器中访问 <{$siteurl}>site-verification.html，确认上传成功。
4. 点击下面的“验证”。
为保持已进行过验证的状态，即使成功通过了验证也请不要删除该 HTML 文件。

向您网站的首页添加元标记
如果您可以修改自己网站的 HTML 编码，则可以使用此选项。		 操作说明：

1. 复制下面的元标记，将其粘贴至您网站的首页。它应该位于第一个 <body> 部分之前的 <head> 部分中。

<meta name="site-verification" content="<{$token}>" />
使用示例
<html>
<head>
<meta name="site-verification" content="<{$token}>" />
<title> 我的标题 </title>
</head>
<body>
网页内容
</body>
</html>

2. 点击下面的“验证”。

为保持已进行过验证的状态，即使成功通过了验证也请不要删除该元标记。
